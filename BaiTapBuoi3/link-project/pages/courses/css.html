Học khóa HTML

Khóa học CSS (Cascading Style Sheets) tại F8 sẽ biến những dòng HTML khô khan thành các giao diện đẹp mắt. Bạn sẽ học từ cơ bản như màu sắc, font chữ đến các kỹ thuật layout hiện đại.

Trọng tâm khóa học: Box Model, Position, Flexbox, CSS Grid và cách dựng layout Responsive để website hiển thị tốt trên mọi thiết bị (Mobile, Tablet, Desktop).

Đặc biệt, bạn sẽ được thực hành cắt PSD sang HTML/CSS thông qua dự án "The Band" huyền thoại.
