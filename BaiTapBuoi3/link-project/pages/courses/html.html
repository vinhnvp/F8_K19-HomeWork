Học khóa CSS

Trong khóa học này, bạn sẽ làm quen với HTML (HyperText Markup Language), ngôn ngữ đánh dấu siêu văn bản giúp tạo nên bộ khung xương vững chắc cho mọi website.

Nội dung bao gồm: Cấu trúc tệp HTML chuẩn, các thẻ Meta, Semantic tags (thẻ ngữ nghĩa), Form và cách tối ưu SEO on-page ngay từ những dòng code đầu tiên.

Bạn sẽ hiểu rõ sự khác biệt giữa thẻ Block và Inline, cũng như cách sử dụng các thuộc tính (Attributes) để điều khiển hành vi của phần tử.
